1. To turn on Tax-Inclusive pricing: From the Manager, navigate to Settings > App Settings> and select Include Vat. Press Save

2. Table Service: If you do not offer any table service, consider creating a Bar type table in the Floor Planner if you simply want to punch in orders without managing a table. Read more about these different table types in the User Guide.

3. Tax: It is important to note that there is no global tax rate you set for your whole account. The appropriate tax rate can be set while creating a product category or by setting it individually in each product if necessary.

4. Takeout/Delivery: Create takeaway and delivery type tables in the Floor Planner. They allow you to create a table that allows for multiple orders at the same time while also keeping track of the customer takeout/delivery information on each order. You can also offer alternate prices and tax rates for your items if they are either takeout, delivery, and regular orders. Read more about this in the User Guide.
